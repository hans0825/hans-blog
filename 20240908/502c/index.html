加入 fish 到 shell

透過 Homebrew 安裝完 fish 後還不能使用，因為沒把 fish 添加到 shell 列表裡。

sudo vim /etc/shells

在列表末尾加上 /usr/local/bin/fish

which fish 可以查看到 fish 被安裝在哪

切換到 fish shell

chsh -s /opt/homebrew/bin/fish

輸入密碼重新開啟終端機後，可透過 fisher 指令操作。

也可以透過下列指令切回到 bash：

chsh -s /bin/bash
